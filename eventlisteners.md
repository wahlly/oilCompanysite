# EVENT LISTENERS.    
    EVENT LISTENERS are some certain functions that wait for the specified event to happen and then go on to implement what has been stated in the function block. Event Listeners also make our web pages more interactive. Event Listeners is achieved by using a method called addEventListener(event, function); the event and the function in which entails what happens during the event, while the event is the is the event handler that is being called.     
    Examples of Event Handlers;
1. onClick  -   when a user clicks on an item.
2. mouseOver  -   when a user place his mouse/cursor     over an item.
3. onLoad   -   when the page loads.
4. mouseOut -   after taking the mouse/cursor away from the item.
5. onFocus  -   when a user put the cursor in a form field.
